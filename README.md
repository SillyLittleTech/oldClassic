# OldClassic - Retro Website Template
A nostalgic, retro-style personal website template inspired by 90s/2000s web aesthetics.

---

## Structure

`/sections/` contains all the data for the different parts of the website <br>
`/data/` contains specifically data for the "Games" section. <br>
`/assets/` has all the images and they are placed in `index.html`, along with other things like scrolling text. 

## Getting Started

1. Replace "Your Name" throughout the site with your actual name
2. Update the content in `/sections/` with your personal information
3. Add your own images to `/assets/` and update references in `index.html`
4. Customize games/projects in `/data/games.json`
5. Update contact information in `/sections/contact.html`

## Customization

Feel free to modify colors, fonts, and styles in `/css/styles.css` to make it your own!
